万维网


网络主要由 HTML 文档组成。 HTML代表 Hypertext Markup Language (超文本标记语言)，是浏览器用来解析网页的语言。网络组件包括客户端（client，你的电脑和浏览器）、互联网（Internet）和服务器（servers）。客户端和服务器通过超文本传输协议（HTTP）进行交互。

基本 HTML 元素


HTML 元素（htmlL Elements）用于指示网页浏览器如何在网页中显示内容。
大多数HTML元素由开始标签（Olpening tag）、以及它们中间的内容（content）组成。

HTML 属性

一些元素在开始标签中，含有我们称之为属性（Attributes）的附加值。属性为浏览器提供额外的信息。例如，链接标签使用 href 属性来定义要链接的 URL。
